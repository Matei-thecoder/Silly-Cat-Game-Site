Description

This game has been made by the author for his girlfriend and future wife. It is a basic runner game with the main character being a cat. There are 5 levels, and the maximum score is 17 (his girlfriend age at the time of the development). The game has been developed in Unity. Enjoy! Disclaimer: This game is designed for Windows PCs x64.

How To Use

The cat can be moved to the left using the "D" button and to the right using the "A" button. The cat can jump when the "W" button is touched. You can also rotate the cat using the left and right arrows. Disclaimer: When starting a level, first make the cat jump, or use the rotate buttons.

How to download

Download the RAR file from the above, extract it, then run the MyFirstProject(1).exe file. Disclaimer: Windows may consider the file dangerous. When running the .exe file ,if a warning window appears, click "Keep File anyway", and then click "Run anyway".

Bugs

It is one of my first Unity Games. I dont deny the possibility that there are bugs in the game. I have found some myself, one being that the jump button doesnt always work. I am working on it. In the mean time, enjoy the game as it is and exploit it how you like. It is supposed to be silly.
